# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to understand passenger characteristics and identify factors that influenced survival. Using Python and popular data analysis libraries, the dataset is cleaned, explored, and visualized to uncover meaningful insights.

---

## 🎯 Objective

The objective of this project is to:

* Explore the Titanic dataset using Exploratory Data Analysis (EDA).
* Clean and preprocess the data.
* Understand the distribution of different features.
* Identify relationships between passenger attributes and survival.
* Create informative visualizations for better data understanding.

---

## 📂 Dataset

* **Dataset Name:** Titanic Dataset
* **Source:** Kaggle
* **File:** `train.csv`

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

---

## 📈 Project Workflow

1. Load the Titanic dataset using Pandas.
2. Display the dataset structure and basic information.
3. Check for missing values.
4. Clean the dataset by handling missing data.
5. Generate descriptive statistics.
6. Explore categorical and numerical features.
7. Create visualizations to understand data patterns.
8. Draw meaningful insights from the analysis.

## 📊 Key Insights

Some important observations from the analysis include:

* The dataset contains missing values that require preprocessing.
* Most passengers belonged to the third passenger class.
* Passenger age is widely distributed across different age groups.
* Survival rates vary significantly based on passenger class and gender.
* Exploratory Data Analysis helps identify important features before building machine learning models.
